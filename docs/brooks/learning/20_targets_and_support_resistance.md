# 目标、支撑和阻力

Brooks 交易必须在入场前理解目标和无效点。支撑阻力不是静态线，而是交易者会改变行为的区域。

## 前高和前低

前高和前低是最常见目标。趋势交易者会看向前高前低，反向交易者也会在这些位置寻找止盈或反转。

价格接近前高前低时，交易者要观察：

- 是否加速突破。
- 是否停顿。
- 是否形成失败突破。
- 是否回测后守住。

## 昨日高低点和日内高低点

昨日高低点和日内高低点常被交易者关注。它们可能成为目标、止损集中区域或失败突破位置。

价格突破这些位置后，如果缺乏跟进，追随突破者容易被困。

## 大整数位

大整数位是视觉和心理目标。很多交易者会在那里止盈、入场或调整仓位。

整数位不是精确支撑阻力。它更像区域，真正重要的是市场到达后如何反应。

## 均线作为支撑阻力

Brooks 可以使用简单均线作为背景参考，但不应把均线当机械信号。

强趋势中，均线附近可能成为回调买入或卖出区域。交易区间中，均线常被反复穿越，信号意义下降。

## 趋势线和通道线

趋势线和通道线帮助交易者观察趋势结构。它们不是为了画出完美线条，而是为了判断市场是否仍沿同一方向推进。

趋势线突破说明结构可能改变；通道线超越可能表示加速或衰竭。两者都需要后续行为确认。

## 区间中轴和边缘

交易区间中轴常是目标，也是方向优势下降的位置。区间边缘是更容易形成交易结构的位置。

区间底部做多的第一目标常是中轴；如果反向强度足够，第二目标可能是另一侧边缘。

## 量度目标

量度目标来自市场常尝试走出相近距离的倾向。

常见来源：

- 交易区间高度。
- 第一段趋势腿。
- 突破前的压缩区域。
- 缺口。
- 旗形或回调结构。

量度目标是目标区域，不是保证。

## Vacuum test

Vacuum test 指价格快速被吸向某个明显目标。它看起来像强趋势，但有时只是因为目标附近订单集中。

到达目标后，市场可能突破、停顿或反转。交易者不能因为 vacuum test 很强就忽略目标处的风险。

## 目标和交易管理

目标决定交易管理方式。目标近、概率高的交易更像 scalp；目标远、需要趋势延伸的交易更像 swing。

如果目标空间不足，就算方向正确，也可能不是好交易。

## 支撑阻力失败

支撑阻力被突破后，如果市场没有跟进并快速回到原区域，突破方向交易者可能被困。这常形成失败突破交易。

支撑阻力真正的价值，不在于预测价格会停在哪里，而在于观察市场到达那里后哪一边失败。
