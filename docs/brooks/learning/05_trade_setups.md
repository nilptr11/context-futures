# 交易 setup

setup 不是形态名称，而是完整交易想法。一个 setup 至少包含市场环境、方向理由、触发、失效点、目标和管理方式。

## 趋势延续

趋势延续交易试图顺着当前控制方入场。典型场景是强趋势或通道中出现回调，反方尝试失败，原趋势恢复。

常见形态语言包括 H2、L2、楔形回调、双测试回调和旗形。

判断重点：

- 市场是否仍处于趋势或有效通道。
- Always In 方向是否仍清晰。
- 回调是否没有破坏趋势结构。
- 反方是否尝试失败。
- 目标是否至少到前高、前低或量度目标。

主要风险是把交易区间中的普通反弹误判成趋势延续。

## 突破延续

突破延续交易试图跟随市场离开旧价格区域。好的突破不只是穿过某个价格，而是显示市场接受新价格。

判断重点：

- 突破是否强。
- 是否有连续性和跟进。
- 回踩是否守住突破点。
- 反方是否无法快速收回突破。
- 目标空间是否足够补偿较差入场价格。

突破延续通常概率较高，但风险回报可能较差。交易者要避免在没有跟进的突破中追高或追低。

## 区间 fade

区间 fade 交易试图在交易区间边缘反向入场。它依赖市场仍在围绕公平价格交易，而不是已经接受突破。

判断重点：

- 区间是否真实存在。
- 价格是否接近区间边缘，而非中部。
- 突破或边缘测试是否缺乏跟进。
- 是否出现反向强度。
- 目标到中轴或另一侧边缘是否足够。

区间 fade 的常见形态是失败突破、双顶、双底和边缘反转。

主要风险是把成功突破误判成失败突破。

## 反转尝试

反转尝试交易试图捕捉趋势失控后的反向运动。它比趋势延续更难，因为交易者必须证明原控制方正在失去优势。

常见背景包括成熟趋势、高潮、楔形、最终旗形、趋势线突破和回测失败。

判断重点：

- 原趋势是否足够成熟。
- 是否出现双边交易增加。
- 是否有趋势线突破或关键结构破坏。
- 回测是否失败。
- 反向目标是否足够。

主要风险是过早逆势。多数看起来像反转的结构，最终只是趋势中的回调。

## 突破模式交易

突破模式交易发生在市场压缩、方向未定时。交易者不应提前假设方向，而应等待突破质量和跟进。

常见背景包括三角形、窄幅震荡、连续内包 K 线和收敛波动。

判断重点：

- 压缩是否足够清晰。
- 突破是否强。
- 突破后是否有跟进。
- 首次突破失败后，反向是否形成更强信号。
- 假突破是否困住了一方交易者。

突破模式的核心是等待市场表态。
