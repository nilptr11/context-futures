# 交易者类型和双边理由

Brooks 的一个重要思想是：同一位置可以同时有多头和空头的合理理由。理解不同交易者的目标和管理方式，有助于避免只看自己方向。

## 突破交易者

突破交易者在市场离开旧价格区域时入场。他们买的是动量和概率。

他们希望看到：

- 强突破 K 线。
- 后续跟进。
- 回调守住突破点。
- 反方无法快速收回。

突破交易者的风险是入场价格差。如果突破没有跟进，他们容易被困。

## 回调交易者

回调交易者等待趋势中的反向运动结束，然后顺势入场。他们希望获得更好价格，同时仍跟随主要方向。

他们关注：

- 趋势是否仍有效。
- 回调是否没有破坏结构。
- 反方是否失败。
- 是否出现 H2、L2、楔形回调或双测试。

他们的风险是把反转误判成回调。

## 区间交易者

区间交易者在交易区间中买低卖高。他们更关注位置和目标，而不是强方向。

他们希望看到：

- 清晰区间。
- 价格接近边缘。
- 突破失败或边缘测试失败。
- 目标到中轴或另一侧边缘。

他们的风险是遇到真正突破。

## 反转交易者

反转交易者试图在趋势失控后站到反方向。他们通常承担更低概率，但希望获得更大回报。

他们需要更多证据：

- 趋势成熟。
- 高潮或楔形。
- 趋势线被突破。
- 回探失败。
- 双边交易增加。

他们的风险是过早逆势。

## Scalper 和 swing trader

Scalper 关注较小目标和较快退出。Swing trader 关注较大目标和更长持有。

同一位置，scalper 可能在区间边缘做反向小目标，swing trader 可能等待突破后持有大目标。两者都可能有理由，差别在概率、目标和管理方式。

Scalp 通常要求更高执行密度、更快判断和更稳定的成交质量。Swing 更强调目标空间、结构无效点和耐心。如果频繁把 scalp 当 swing、或把 swing 管成 scalp，最容易让原本合理的交易数学失效。

## 管理复杂度

交易方式越复杂，对实际风险控制的要求越高。

高复杂度管理包括：

- 逆势限价入场。
- 分批入场和加仓。
- 在交易区间中承受更深不利波动。
- 用小目标高频退出。

这些方式不是错误，但必须有清晰的最大风险、退出逻辑和目标空间。否则，复杂管理会把原本有优势的交易变成不可控风险。

## 为什么对手愿意成交

每次入场都应该问：我的对手为什么愿意在这里成交？

如果你买入，卖方可能是：

- 获利了结的多头。
- 认为价格过高的空头。
- 在交易区间顶部卖出的限价交易者。
- 认为突破会失败的反向交易者。

理解对手方，不是为了知道真实身份，而是为了保持双边思维。
