# Brooks 价格行为学学习路线

本目录是 Brooks 价格行为学的学习材料。这里不讨论任何交易系统、程序实现或实验结果，只解释价格行为阅读本身。

## 第一阶段：核心框架

建议先按顺序阅读：

1. `00_overview.md`：总览，理解 Brooks 方法为什么不是形态清单。
2. `01_market_cycle.md`：趋势、通道、交易区间、突破、突破模式和高潮。
3. `02_context.md`：Always In、控制权、位置、支撑阻力和磁吸目标。
4. `03_order_flow.md`：止损单市场、限价单市场、失败、跟进和被困交易者。
5. `04_patterns.md`：H1/H2/L1/L2、楔形、双顶双底、三角形、最终旗形和主要趋势反转。
6. `05_trade_setups.md`：顺势延续、突破延续、区间 fade、反转尝试和突破模式交易。
7. `06_trade_management.md`：入场、止损、目标、scalp、swing、加仓减仓和 Trader's Equation。
8. `glossary.md`：术语表。

## 第二阶段：专题深化

核心框架之后，再读这些专题：

1. `07_time_of_day.md`：不同时段的交易行为。
2. `08_multi_timeframe.md`：多周期关系。
3. `09_advanced_patterns.md`：高级形态和嵌套结构。
4. `10_trader_types.md`：不同交易者为什么能在同一位置做相反交易。
5. `11_risk_and_psychology.md`：实际风险、连续亏损和心理纪律。
6. `12_bar_by_bar_reading.md`：逐根 K 线阅读方法。

## 第三阶段：细节深化

专题深化之后，再补这些细节：

1. `13_bar_types.md`：K 线类型、信号 K 线、入场 K 线和失败信号。
2. `14_trends_and_channels_deep_dive.md`：趋势和通道细分。
3. `15_trading_ranges_deep_dive.md`：交易区间细分。
4. `16_reversals_deep_dive.md`：反转体系。
5. `17_second_entries_and_traps.md`：二次入场和陷阱。

## 第四阶段：案例和目标深化

细节框架建立后，再用更具体的场景训练阅读：

1. `18_day_types.md`：交易日类型。
2. `19_open_and_gap.md`：开盘和缺口。
3. `20_targets_and_support_resistance.md`：目标、支撑和阻力。
4. `21_trade_management_examples.md`：交易管理示例。
5. `22_bar_by_bar_examples.md`：逐根 K 线阅读示例模板。

## 阅读原则

不要把 Brooks 学成形态识别。每个形态都要回到五个问题：

1. 当前市场处于什么循环？
2. 多空哪一边控制市场？
3. 价格在什么位置？
4. 哪一边可能失败或被困？
5. 这笔交易的概率、风险、回报和管理是否合算？

如果无法回答这些问题，就算形态名称正确，也不是完整的价格行为阅读。
